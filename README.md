# Todo App

Una Todo App sviluppata con **React**, **TypeScript** e **Vite**, realizzata come **progetto di studio** con l'obiettivo principale di comprendere come utilizzare **Cursor** come supporto allo sviluppo frontend.

Durante il progetto, Cursor è stato utilizzato per assistere nella scrittura del codice, nel refactoring e nell'analisi dei componenti, mantenendo sempre il controllo manuale sulle scelte progettuali e sull’implementazione finale.

## Funzionalità

- Gestione dei todo con stato completato
- Sistema di tag
- Filtri per stato (All / Active / Completed)
- Filtro per tag combinabile
- Interfaccia minimale con Tailwind CSS (via CDN)

## Tecnologie utilizzate

- React
- TypeScript
- Vite
- Tailwind CSS (CDN)

## Avvio del progetto

```bash
npm install
npm run dev
```

## Note

- I dati sono gestiti solo in memoria tramite stato React.
- Il progetto è pensato come esercizio di apprendimento e sperimentazione.
